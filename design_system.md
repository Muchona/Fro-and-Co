# Design System: Fro & Co (Dark Executive)

## Status
**Agent A (UI):** WORKING
**Agent B (Animation):** STANDBY

## Core Aesthetic
**"Modern Luxury meets Industrial Coffee"**
- **Vibe:** Dark, textured, premium, mysterious.
- **Glassmorphism:** Heavy blur (15px), subtle borders (#ffffff10).
- **Texture:** Persistent grain overlay.

## Color Palette
### Primary
- **Charcoal**: `#0F0F0F` (Main Background)
- **Espresso**: `#121212` (Card Backgrounds, Sections)

### Accents
- **Toasted Gold**: `#C4A484` (Highlights, Buttons, Text Accents)
- **Creamy White**: `#F5F5F5` (Primary Text)

## Typography
- **UI/Body**: `Inter` (Clean, legible, modern)
- **Technical/Pricing**: `JetBrains Mono` (Data, details, precision)

## Animation Protocols (Agent B)
- **Hero**: Scroll-triggered Image Scale (Zoom) + Text Fade Out.
- **Icons**: GSAP Staggered Float + Fade.
- **Micro**: "Neon Aura" hover states on buttons.
- **Global**: Smooth Reveal on scroll (Slide Up + Fade).

## Layout
- **Mobile**: Vertical Stacks, Reduced Grain.
- **Desktop**: Grid layouts, ample negative space.
